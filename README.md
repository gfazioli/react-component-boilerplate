# React Component Boilerplate

A minimal boilerplate useful to create a React Component [React](http://facebook.github.io/react/index.html)

# Usage

Onnce clone the repo

* remove the `.git` folder
* rename `mycompnent` filename
* rename `MyComponent`
* run `yarn`
* run `npm run watch`
